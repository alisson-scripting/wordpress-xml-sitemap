## Description

Combined XML Sitemap generates a dynamic XML sitemap for:

- Posts
- Pages
- Categories
- Images

## Installation

1. Copy the plugin files to `wp-content/mu-plugins/`.
2. If the plugin is located in a subfolder, place the loader file directly in `mu-plugins`.
3. MU plugins do not need to be activated in the WordPress admin.
